# Product Overview

Placement Mail Analysis System - An AI-powered system for analyzing placement-related emails and documents.

## Purpose

Processes and analyzes placement emails using natural language processing, document extraction, and AI models to extract insights and structure information from recruitment communications.

## Key Capabilities

- Email processing via Gmail API
- PDF and document parsing
- Natural language processing with spaCy and NLTK
- Vector embeddings and semantic search with ChromaDB
- AI-powered analysis using Google Generative AI
- Data export to Excel formats
